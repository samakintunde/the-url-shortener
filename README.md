# URL Shortener

Web: https://url-sh.fly.dev/

API: https://url-sh.fly.dev/api

Health: https://url-sh.fly.dev/health

## Install dependencies

Installs dependencies of the web client

```sh
make install
```

## Start development environment

```sh
make dev
```

## Build app

```sh
  make build
```
